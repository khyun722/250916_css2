반갑습니다, 성장하는 백엔드 개발자입니다.
안정적이고 확장 가능한 시스템을 구축하는 것에 큰 흥미를 느끼며,
대규모 트래픽을 다루는 경험을 쌓고 싶습니다.

🛠️ 기술 스택

💻 포트폴리오

여행 추천 프로젝트

Spring Boot 기반으로 사용자 맞춤형 여행지 추천 API를 개발했습니다.
JPA를 활용해 데이터를 관리하고, 사용자 리뷰와 평점을 분석하여
추천 알고리즘의 정확도를 높이는 데 기여했습니다.

재능 매칭 프로젝트

실시간 재능 매칭 및 채팅 기능을 중심으로 한 플랫폼을 구축했습니다.
WebSocket을 이용해 사용자 간의 즉각적인 소통을 구현하고,
안전한 서비스 이용을 위해 예약 및 결제 시스템을 설계했습니다.

AWS 인프라 프로젝트

대규모 트래픽에 대응 가능한 클라우드 인프라를 설계하고 구축했습니다.
AWS EC2, S3, RDS를 기반으로 시스템을 구성하고,
Load Balancer와 Auto Scaling을 적용하여 서비스의 가용성과 확장성을 확보했습니다.
